Introducing Subsatoshi

Subsatoshi aims to be the reference open-source implementation of the Value Distribution Protocol. In a nutshell, it enables the distribution of value across a large number of related stakeholders, powered by very small off-chain cryptocurrency transactions, known as microtransactions.

How Small is Small?

Very small. We estimate that for our main use case we will need a granularity of up to 0.0000000000000001 (that's 16 decimals, which is twice as many decimal places as Bitcoin currently supports).

What Cryptocurrencies are Supported?

At first we will focus on bitcoin, but our goal is to support as many altcoins as possible.
